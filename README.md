# gen

A simple tool for generating common random data

# Available commands

- Random UUID

```sh
$ gen uuid
```

- Random CPF

```sh
$ gen cpf
```

- CPF validator

```sh
$ gen cpf validate 06248751307
```

- Name generator

```sh
$ gen name
```

- Email generator

```sh
$ gen email
```

# Build

```sh
$ make build
```
